# 📖 个人知识库使用指南

## 快速开始

### 1. 启动服务器

```bash
cd "/Users/Zhuanz/Desktop/项目文件/2026.06.22 my blog/"
node server.js
```

启动后访问：**http://localhost:8080**

---

## ✍️ 发布文章（两种方式）

### 方式一：富文本模式（推荐）—— 像微信一样粘贴

1. 在首页点击右上角 **"✍️ 发布文章"** 按钮
2. 输入文章标题
3. 切换到 **"🎨 富文本模式"** 标签
4. **直接从微信公众号编辑器复制已排版好的内容，粘贴到编辑框里**
5. 点击 **"👁️ 预览"** 查看效果
6. 点击 **"💾 保存发布"**，文章就发布成功了！

### 方式二：Markdown 模式 —— 手写文章

1. 在首页点击右上角 **"✍️ 发布文章"** 按钮
2. 输入文章标题
3. 在 **"📝 Markdown 模式"** 标签下输入内容
4. 支持语法：
   - `# 一级标题` `## 二级标题` `### 三级标题`
   - `**粗体**` `*斜体*`
   - `- 无序列表` `1. 有序列表`
   - `> 引用文字`
   - `[链接文字](URL)`
   - `![图片描述](图片路径)`
5. 预览满意后点击 **"💾 保存发布"**

## 📱 手机访问

1. 查看本机 IP：
   ```bash
   ipconfig getifaddr en0    # WiFi
   ```
2. 手机上访问：`http://你的IP:8080`

> ⚠️ 手机和电脑必须在同一个WiFi下

## 💡 常见问题

**Q: 粘贴微信文章后格式乱了怎么办？**
A: 富文本模式会自动把微信的HTML格式转换成Markdown，大部分情况没问题。如果个别格式不对，可以手动在Markdown模式下微调。

**Q: 图片怎么处理？**
A: 微信文章里的图片如果粘贴进来，会自动转成Markdown的图片格式。如果需要本地显示，可以把图片放到 `posts/` 目录下，用相对路径引用。

**Q: 怎么关闭服务器？**
A: 在终端按 `Ctrl + C`。下次用 `node server.js` 重新启动。

**Q: 可以编辑已发布的文章吗？**
A: 目前需要先删除旧文件再重新发布。后续版本会加上编辑功能。
